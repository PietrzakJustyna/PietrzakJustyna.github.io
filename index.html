Here is a description of some of my projects

BookFreaks app

This is an app that was created as a final project during my Python course. It is a web crud application built with Python & Django with some JavaScript & JQuery. I am using PostgreSQL database to store the data. Tests are written in unittest.
App has three levels of permissions - Not Logged User, Logged User & Admin - which give access to different options & views. User that is not logged in can see the list of books and authors. Logged in user gains access to rate books & authors and to mark books or authors as their favourites. Admin can add new elements to database & has access to admin panel.

Language Predictor

In this project I wanted to combine languages & programming. Following a tutorial and using data from wikipedia api I built a model in Keras that with around 90% correctnes can asses if the word given is in Polish or English. The simple UI part is written in Flask, and app itself is deployed on Heroku.
